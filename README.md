✅ تحليل SSL/TLS متقدم (CVE-2023-0001)

✅ مسح دليل ذكي مع دعم Multithreading  
✅ تقارير احترافية (HTML/PDF)  
✅ تكامل كامل مع Nuclei Templates  
✅ دعم البروكسي و User-Agent مخصص  
✅ نظام تسجيل أخطاء متقدم
